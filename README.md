# Walnut 同步工具: wnsync

一个与 walnut 服务器端同步数据的客户端。它的任务是:

1. 将一个本地目录同步到服务器端
2. 将一个服务器端目录同步到本地

由于考虑到用户权限的问题，*wnsync* 需要保留用户的登录信息

# 界面:登录

启动程序后就是登录界面

```
              WNSYNC
------------------------------------

   Server   : ______________

   UserName : ______________

   Password : ______________

             Do Signup

             [x] Keep login next time
```

# 服务器提供的接口

```
登录 : /u/do/signup
-------------------------------------------------------
获得同步列表 : /sync/check
    
    服务器会分配一个临时文件夹，过期时间为1天。
    这个文件夹的 ID 将作为本次同步的 ID

-------------------------------------------------------
上传本地文件 : /sync/upload

    不到10M的小文件，统统打成压缩包包上传
    一个压缩包最大不超过 1G
    大文件随后上传

-------------------------------------------------------
下载远端文件 : /sync/download
    
    服务器会把在同步文件内把需要下载的文件打包。
    不到10M的小文件打成压缩包
    一个压缩包最大不超过 1G
    大文件单独放置。 这些文件的索引数据会在 /sync/check
    时发送给客户端。客户端在需要下载的时候，指定某一项
    进行下载。支持 http 断点续传

-------------------------------------------------------
结束同步 : /sync/done

    服务器将会根据同步设置，将上传包里的文件解开，
    copy 到目标位置。同时也会删除需要删除的文件

```
